- `sign(request, identity, opts={})`:
  1. Defensive check: throw if `!identity.signatureAgentUrl.startsWith("https://")`
  2. Set `request.headers["Signature-Agent"] = '"' + identity.signatureAgentUrl + '"'` (RFC 8941 quoted-string) — MUST be before signatureHeaders call (Pitfall 6)

  7. Mutate `request.headers["Signature"]` and `request.headers["Signature-Input"]` (mirror Python signer behavior)
  8. Return `{ signature, signatureInput, signatureAgent }` (camelCase per D-09 carry-forward)
